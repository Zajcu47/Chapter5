Module 2: Plan for Maintenance of Computer Systems and Networks

Plan for Maintenance and Repair

Welcome to the Module “Maintain and Repair Computer Systems and Networks”. This module contains information and activities for you to complete. It provides students with skills to diagnose, troubleshoot, and repair common hardware and network problems.

You will learn to apply preventive maintenance, fault diagnosis, and systematic troubleshooting techniques to ensure efficient performance.

Objectives

  • • Identify parts and tools for maintenance and repair.
  • • Perform step-by-step maintenance procedures.
  • • Commit to regular maintenance for longevity.

Introductory Activity

Pre-Diagnostic Test/Assessment: Skill Test

Instructions: Choose the most appropriate tool for each scenario.

Scenario 1: Virus Scan & Malware Removal

Fix a slow-running desktop computer affected by a virus. Perform a virus scan, clean out temporary files, and remove malware.

Scenario 2: Flickering Monitor

Inspect the hardware, verify display cables, and ensure the graphics card is properly seated.

Scenario 3: Laptop Won’t Power On

Check if the power supply is working, verify the battery, and inspect the motherboard for visible damage.

Scenario 4: Upgrading RAM

Upgrade the RAM in a desktop computer to improve performance.

Scenario 5: Network Troubleshooting

Test the ethernet cable, router, and network card.
